4)  Show me all notes that their titles do not equal "Backup"

**Trilium DSL query:**
```
note.title != 'Backup'
```
- Search Structure
```json
{
  "searchCriteria": [
    {
      "property": "title",
      "type": "noteProperty",
      "op": "!=",
      "value": "Backup",
      "logic": "AND"
    }
  ]
}
```
